# 🎓 HCP2 University Website

A responsive **University Website UI** built using **HTML and CSS**.
This project demonstrates a modern educational institution website with multiple pages such as Home, About, Courses, Blog, and Contact.

The website design focuses on clean layout, responsive structure, and user-friendly navigation.

---

## 🚀 Features

* Responsive website layout
* Modern university landing page design
* Multiple webpages
* Navigation menu
* Course section
* Blog section
* Contact page
* Image-based campus showcase

---

## 🧩 Pages Included

* **Home Page** – Main landing page with university introduction
* **About Page** – Information about the university
* **Courses Page** – Displays available courses
* **Blog Page** – Educational blog section
* **Contact Page** – Contact information and form

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Responsive Layout Design
